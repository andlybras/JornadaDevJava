## Trilhas de Estudo  
  
Este diretório armazenará minhas trilhas de estudo, englobando qualquer tecnologia, conceito ou ideia que eu esteja aprendendo. Por exemplo, a primeira trilha é o diretório `01 - Livro Core Java Volume I - Fundamentals (Cay. S. Horstmann)`, que conterá notas de estudo, resoluções de exercícios, etc., referentes ao livro que dá nome à pasta.  
  
Dessa forma, será possível documentar a evolução da minha jornada de estudo em Java, seja pela quantidade de trilhas desenvolvidas, seja pelos tipos de fontes consultadas.
